# Data Access Instructions

The original VIIRS Nighttime Lights dataset (2013–2017, Asia) is too large to be stored in this repository.

You can download the full dataset from:
https://eogdata.mines.edu/products/nighttime_lights/

Steps:
1. Visit the link above.
2. Navigate to VIIRS Nighttime Lights (VNL V2).
3. Download annual composites for 2013–2017.
4. Place the downloaded files into `data/raw/`.
